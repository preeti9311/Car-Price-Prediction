# 🚗 Used Car Price Prediction

## 📌 Overview

This project focuses on predicting the selling price of used cars using Machine Learning techniques. The model is trained on a dataset containing information about used cars sold in India and their key features.
The objective is to estimate a car's market value based on factors such as manufacturing year, fuel type, transmission type, ownership history, and kilometers driven.

---

## 🎯 Problem Statement

Determining the correct selling price of a used car can be challenging due to multiple influencing factors.

This project aims to build a machine learning model that predicts the selling price of a used car based on its features. Such predictions can help:

- Sellers determine a fair market price.
- Buyers evaluate whether a car is reasonably priced.
- Automobile marketplaces provide accurate pricing recommendations.

---

## 📊 Dataset Information

- Source: CarDekho Dataset
- Records: 15,411
- Features: 13 Columns
- Target Variable: Selling Price
